Intro to our project

List Llama is a grocery list Web App created by Anders Long, Chase Prochnow, Natalie Ruckman and Trevor Ives for UNL's Fall 2020 CSCE 361 Class. The web app was created using the asp.net technology stack, SQL and Azure to build, store and host the project. I worked mostly on the backend except for several times where I implemented small features to the ui. You can take a look at the project repo to see all the hardwork Group 5 and I put into our project!
